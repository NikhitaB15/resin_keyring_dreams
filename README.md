## üé® Key Features
*   **Stunning UI**: Custom "Soft Rose" & "Gold" color palette with glassmorphism effects.
*   **Dynamic Product Catalog**: Real-time fetching of products from Supabase.
*   **Smart Search & Filtering**: Instant search by title/description and category filtering.
*   **Interactive Cart**: Sliding "Bag" drawer with quantity management.
*   **Location-Based Shipping**: Auto-detects user location (Indian Pincodes) to calculate shipping costs (Standard vs. Metro).
*   **Admin Dashboard**: Protected route for adding, editing, and deleting products directly in the live app.
*   **Smooth UX**: "Heartbeat" loading screens, toast notifications, and micro-animations.

## üõ†Ô∏è Setup & Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/StartYourDreams/resin_keyring_dreams.git
    cd resin_keyring_dreams
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Environment Variables**
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run Locally**
    ```bash
    npm run dev
    ```

5.  **Build for Production**
    ```bash
    npm run build
    ```

## üì¶ Database Schema (Supabase)
The app uses a single `products` table:
```sql
create table products (
  id bigint generated by default as identity primary key,
  title text not null,
  price numeric not null,
  original_price numeric,
  category text,
  image text,
  description text,
  tags text[], -- Array of strings
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
```
